程序猿从入门到放弃

  • 随笔

成为程序猿之路

成为一名程序猿的方法是多样的，路途是坎坷的，方向是繁多的，前途是光明的。哇咔咔，我竟然会排比，看来写一篇高中作文能拿个高分。

一、有一个电脑

  • Windows：
    上网、影音、办公、敲代码、打游戏
  • Mac OS(OS X):
    上网、影音、办公、敲代码

二、练习打字

上图榜单入口：全宇宙打字比赛

wpm(words per minute) : 单词/每分钟

kpm(key per minute) : 键/每分钟

常用的练字客户端(排名不分先后)：

Mac OS(OS X): typist

Windows : 金山打字通

刚开始练习打字速度在20wpm左右不要气馁，最主要先熟悉姿势和手位，再慢慢练，然后朝着40、80、120、160wpm等更高的方向努力，练到80左右就算个正常水平吧。
以我单身二十多年的手速是60wpm左右，看来用QQ或者微信提升感情是你们练字的绝佳机会。

三、英语单词

计算机常用英语单词(xls)

代码都是英文的,即使英文不好，也要把常用的单词都记住。
英语好一点的，阅读阅读英文文档，写点双语博客，会觉得自己屌屌的。

四、程序猿基础

一个人想要成为优秀程序员，就必须有坚实的基础。核心理念的理解会帮助你用最好的方法设计和实施出最完美的方案。

  • 掌握一种编程语言。学习任意一种主流的编程语言，例如C/C++、java或者C#。学好一种语言，再学其他的语言就简单的多了。

    C/C++练手工具：codeblocks for windows
